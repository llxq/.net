简介：
	-> JavaScript代码在浏览器中运行,做出更流畅、优美的页面效果,增强用户体验
	-> 他与java是完全不同的东西,只是名称类似而已
	-> 写在 <script></script> 标签中
	
alert(里面填写参数)		这个效果是弹出一个对话框

-> JavaScript放置位置：
	-> head部分：用于声明变量、函数、类型、为事件绑定处理函数
	-> body部分：调用脚本执行
	-> 外部脚本：用于定义函数、类型
		-> 将代码封装到一个扩展名为js的文件夹中,然后在需要的地方引用,完成一次定义,多处引用的效果,简
		   化了代码的维护。
		-> 在文件中不需要写 <script></script> 标签
		
-> 如何添加外部js脚本：
	<script src="文件路径"></script>
	
-> 我们的HTML文件是从上至下执行的

-> 语法：
	-> 大小写敏感
	-> 弱类型语言(弱类型语言声明变量不需要明确规定变量需要占多大内存,只有当这段代码运行的时候才知
				  道占多大内存,这种就是弱类型语言)
				  -> 先赋值int类型,然后又赋值string类型都行,赋值什么都行
		-> C#是强类型语言,强类型语言就是你声明一个变量必须要有类型
	-> 分号结尾
	-> 字符串推荐使用单引号,也可以使用双引号

-> 获取时间：
	-> var date=new Date();
	-> alert(tostring());			//获取世界时间,会提示当前时区
	-> alert(toLocaleString());		//获取当前时区时间,转换为字符串
	
-> 一般我们会把HTML代码和js代码分离,不会混合写	
	
-> 在js中,定义的类的名字首字母为大写,方法的首字母为小写
	
-> 声明方法：(写在script标签里面)
	function 方法名(参数)
	{
		方法体
	}
在方法体中加上return false; 可以让a标签不进行跳转,但是在调用的时候得加上一个return 方法名();
	
如果我们不想让a标签跳转,可以将他的href地址写成一段脚本：
	<a href="javascript:方法名()"></a>
	
-> 点击事件：onclick
