-> DOM简介：
	-> DOM：文档对象模型
	-> 是W3C组织推荐的处理可扩展标记语言的标准编程接口
	-> 将所有标记加载到内存,以树形结构处理
	-> 万维网联盟(World Wide Web Consortium,W3C),又称W3C理事会。1994年10月在麻省理工学院计算机科学
		实验室成立。创建者是互联网的发明者蒂姆.博纳斯.李
	-> 每种开发语言对于这套规范都有不同的具体实现
	
-> DOM对象拥有的成员包括：事件、属性、方法
-> 使用JavaScript进行DOM操作,主要包括两个方面：
	-> Browser对象,即BOM,指window.***     //操作窗口的
	-> HTML DOM对象,即狭义的DOM,指document.***	//用于操作<html></html>里面的
	
->窗口对象 window 
	-> window是DOM的顶级对象,代表当前浏览器窗口
	-> 建议：标准访问语法为 "window.***" ,可以省略window,这样可以减少文件的字节数
	
-> 方法：对话框
	-> alert()：提示对话框
	-> confirm()：确认对话框,返回一个boolean类型的值,点击确定返回true,点击取消返回false
	-> prompt()：用于显示提示用户输入的对话框,如果点击确定则返回输入值,如果点击取消则返回null值

-> 方法：定时器
	-> setinterval(code,delay),表示每隔delay毫秒执行一次code的代码
	-> code代码需要加入引号中
	-> 如果code较多可以封装到一个函数中,这里写函数的名称,不需要加引号
	-> 使用建议：在一个页面中,尽量不要启动多个定时器,启用一个就够了
	-> 这个方法返回一个int的值,表示定时器标识,可以用于清楚定时器：clearinterval(intervalid);
	
-> 方法：延迟器
	-> setTimeout(code,delay),表示等待delay毫秒后再执行code代码
	-> code代码需要加入引号中
	-> 如果code较多可以封装到一个函数中,这里写函数的名称,不需要加引号
	-> 说明：code代码执行一次之后不在执行
	-> 他也是返回一个int类型的值,表示标识,可以用于清楚延时器：cleartimeout(标识)
	
-> 属性：重定向 location
	-> 表示记录当前浏览器窗口地址栏信息的对象
	-> 属性href用于设置加载的URL
	
-> 属性：文档 document
	-> 文档对象是窗口对象的属性,用于表示当前窗口中所有HTML构成的一个对象
	-> 用于对标签元素进行操作

-> disable：该属性可以设置是否禁用,true为禁用	

-> 通过修改元素的style属性操作样式
	-> js中的style对象提供了很多属性,对应着css中的相关样式
	-> 这里的写法与css基本相同
	-> 如果在css中有 "-" 的样式,把 "-" 去掉,将 "-" 后面的字母首字母大写 格式如：backgroundColor
	-> 对于某些元素拥有的属性,可能在提示中不存在,直接写就可以了
	-> 设置样式类：dom对象.className='类名称'
-> 特例：float
	-> 普通写法：dom对象.style.float='right'   //float是js关键字,不推荐使用,ff(火狐)不支持
	-> dom对象.style.cssFloat='right'		//标准写法,ie新版本及其他浏览器兼容
	
-> 控制层的显示与隐藏
	-> 通过设置样式display控制显示与隐藏
	-> 值为 block 表示显示
	-> 值为 none 表示为隐藏   (当设置为隐藏的时候,这个元素就不占页面空间了,下面的元素会往上走)
	
-> from对象：
	-> 使用传统方式进行表单提交：点击submit按钮
	-> 事件onsubmit：表示在提交前触发,一般用于进行控件内容的有效性验证
	-> 如果返回false则会停止提交
	-> 如果返回true或者不返回则完成提交
	-> onsubmit事件的作用：可以在转向页面前,进行数据的有效性验证
	-> 取消提交的方式：return false;
	
	-> 问：如果在不点击提交按钮时进行提交呢？
	-> 方式一：调用按钮的click()方法,模拟点击 (使用submit的onclick事件,将事件当做一个方法用,会
			   模拟用户对这个控件的事件的操作)
		-> 就是调用其他按钮的onclick事件,在里面调用submit的click事件则可以触发from的onsubmit方法
	-> 方式二：调用表单的submit()方法,这种提交是不会触发onsubmit事件
		-> 就是调用其他按钮的onclick事件,在里面直接调用form的submit()方法,这个方法不会触发onsubmit
			事件
	
-> 在js中如果将一个事件赋值给另一个事件的话,则直接相等即可,不需要加等于
